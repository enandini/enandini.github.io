Predicting the Presence of Heart Disease¶

By: Eshita Nandini¶

This dataset used is provided by The UCI Machine Learning Repository. In the version of the data I am using, there are only 14 attributes as they are commonly the ones preserved in past studies. Through this notebook, I try to predict the presence of heart disease by understanding the features present and solving a binary classification problem.

The challenge can be found here: https://www.kaggle.com/ronitf/heart-disease-uci

Attribute information:¶

This information was obtained from the Kaggle site:

age: Age in years.

sex: (1 = male; 0 = female)

cp: Chest pain type (Value 1: typical angina, Value 2: atypical angina, Value 3: non-anginal pain, Value 4: asymptomatic)

trestbps: Resting blood pressure (in mm Hg on admission to the hospital)

chol: Serum cholesterol in mg/dl

fbs: (fasting blood sugar > 120 mg/dl) (1 = true; 0 = false)

restecg: Resting electrocardiographic measurement (0 = normal, 1 = having ST-T wave abnormality, 2 = showing probable or definite left ventricular hypertrophy by Estes' criteria)

thalach: Maximum heart rate achieved

exang: Exercise induced angina (1 = yes; 0 = no)

oldpeakST: Depression induced by exercise relative to rest ('ST' relates to positions on the ECG plot)

slope: The slope of the peak exercise ST segment

ca: Number of major vessels (0-3) colored by flourosopy

thal: 3 = normal; 6 = fixed defect; 7 = reversable defect (thalassemia - inherited blood disorder)

target: 1 = has heart disease; 0 = doesn't have heart disease

Here are a few things I will use to understand the performance of this regression:

Classification Accuracy: This is how well the model did at predicting the correct label (target = 1 or target = 0).

Misclassification (Error) Rate: This is how many times the model predicted incorrectly.

Sensitivity (Recall): When the actual value is positive, how often the prediction is correct.

Specificity: When actual value is negative, how often is the prediction correct.

IV. Evaluation + Conclusion¶

It appeared the logistic regression and the random forest yielded the highest Classification Accuracy (84.6%) as well as lowest Error Rate (15.4%). However, in terms of Specificity, the random forest performs better by about 2%, but the logistic model is about 2% more Sensitive.
